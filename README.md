Welcome to the source code for my personal portfolio! This repository contains a modern, interactive, and fully responsive developer portfolio built to showcase my Software Engineering journey, key achievements, technical skills, and projects.

## 🚀 Live Demo
**https://satishs-portfolio.vercel.app**

## ✨ Key Features
- **Modern Tech Stack**: Built with Next.js (App Router), React 19, and Tailwind CSS.
- **Premium Animations**: Leverages Framer Motion for smooth, highly interactive scroll animations, glowing elements, and micro-interactions.
- **Interactive Resume**: A dedicated `/resume` page that beautifully formats and animates professional experience and skills.
- **Dynamic 3D Integration**: Includes `@splinetool/react-spline` for dynamic 3D background capabilities.
- **Accessible UI Components**: Uses Radix UI (`shadcn/ui`) for highly accessible, customizable, and reliable interactive components.
- **Responsive Layout**: Carefully tailored for Desktop, Tablet, and Mobile views.
- **Dark Theme Optimized**: Designed specifically with a striking "Aetheric Noir" premium dark aesthetic.

## 🛠️ Tech Stack & Dependencies
* **Framework**: Next.js 16
* **Language**: TypeScript
* **Styling**: Tailwind CSS
* **Animations**: Framer Motion
* **UI Components**: Radix UI / shadcn/ui
* **Icons**: Lucide React
* **Linting & Formatting**: ESLint, Prettier, Husky (Git Hooks)

## 💻 Running the Project Locally

To test or run this portfolio on your own machine:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/COutSatish/portfolio.git
   cd portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **View the application:**
   Open your browser and navigate to `http://localhost:3000`.
